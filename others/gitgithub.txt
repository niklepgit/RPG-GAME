ΟΔΗΓΟΣ ΓΙΑ ΤΑΛΑΙΠΩΡΗΜΕΝΟΥΣ με το GIT/GITHUB

GIT:
	1)κατεβαζουμε το git απο το site 
	2)φτιαχνουμε ενα φακελο με το ιδιο ονομα που εχει το project που θελουμε να δουλεψουμε.
	3)κανουμε git init
	4)βαζουμε την πηγη απο το online repository που εχουμε στο github: git remote add origin https://github.com/nik123git2/			icecream.git
	5)εκτελουμε git remote -v για να δουμε οτι ολα ειναι οκ
	6)git config --global user.name "Your Name Here"
	7)git config --global user.email "your_email@youremail.com"

	ειμαστε ετοιμοι να δουλεψουμε σε δικο μας project!!

GITHUB:
	1)φτιαχνουμε προφανως λογαριασμο στο github 
	2)στο repository που εχουμε το project παμε settings->collaborators και κανουμε invite τον συναδελφο.
	3)στελνουμε με μηνυμα ή οπως σκατα θελουμε το url με το invitation που βγαζει.
	4)ο συναδελφος παταει accept στο url.

εντολες αν δουλευουμε μονο σε master branch:

git add name
git commit -m "..."
git push origin master
git pull origin master
git status 

HOW TO CACHE THE CREDENTIALS:
1)
$ git config --global credential.helper cache
# Set git to use the credential memory cache

2)
$ git config --global credential.helper 'cache --timeout=3600'
# Set the cache to timeout after 1 hour (setting is in seconds)
